Bienvenido a la API de Prueba (Testing Api)

Esta API es un ejercicio para la materia de Cloud Computing en la cual se incluyen consumos de APIs de terceros mediante métodos GET y POST.

Cada extensión del presente servidor ofrece servicios tanto de consulta como de entretenimiento, incluyendo algunas utilidades para creación de trabajos o prácticas con el uso de APIs

Puede que parezca dificil crear o utilizar métodos en una API, por eso, consulte el archivo README.md del repositorio GitHub, los cuales brindarán instrucciones e información para el uso de las APIs configuradas, así también si le interesan, algunos ejemplos o maneras en las que usted pueda utilizarlos en el archivo de Documentacion.

Todos los métodos deben ser invocados empezando por "localhost:8080" seguido de una de las extensiones soportadas

Extensiones soportadas Nombre Nombre de Sitio de origen (endpoint) Ejemplo
/diasfestivos/{año}/{codpais}; Dias Festivos NagerDate /diasfestivos/2021/US
/clima/{una ciudad} Obtener Clima OpenWheaterMap /clima/mcallen
/entretenimiento/musica/{una banda o cantante} Obtener info música MusicBrainz /entretenimiento/musica/scorpions
/entretenimiento/studioghiblipeliculas Obtener Info Studio Ghibli Ghibli Api idem
/entretenimiento/studioghiblipeliculas/{una pelicula} Info sobre película SG Ghibli Api /entretenimiento/studioghiblipeliculas/Castillo_Ambulante
/entretenimiento/edades/{nombre} Edad de nombres Agify.io /entretenimiento/edades/jessie
